Эта программа служит для демострации лабораторной работы №3 "Наследование. Полиморфизм."
В соответствии с заданием имя файла конфигурации считывается с командной строки.

Для корректного функционирования программы необходимо установить дистрибутив freeglut,
если он не был установлен ранее на Вашем локальном компьютере.

Для установки freeglut рекомендуется выполнить следующие действия:
1)дистрибутив freeglut доступен по ссылке https://www.transmissionzero.co.uk/software/freeglut-devel/
2)в случае работы с MSVC загружается https://www.transmissionzero.co.uk/files/software/development/GLUT/freeglut-MSVC.zip
3)распаковать архив во временную директорию
4)прочитать Readme и следовать инструкции

Далее необходимо настроить MSVC. В свойствах проекта:
1. С/С++ -> общие -> дополнительные каталоги включаемых файлов -> C:\Program Files\Common Files\MSVC\freeglut\include
2. Компоновщик -> общие -> дополнительные каталоги библиотек -> C:\Program Files\Common Files\MSVC\freeglut\lib
3. ОК

Файлы с расширением .dll
В 64-разрядной системе (например, Windows 10) поместите 64-разрядные .dll в папку C:/Windows/SysWOW64, 
в которой хранятся 64-разрядные динамические библиотеки. 
Папка System32, предназначена для хранения 32-разрядных .dll.
